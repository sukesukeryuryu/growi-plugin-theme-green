# growi-plugin-theme-green

A clean green & white GROWI theme inspired by academic lab website aesthetics.

## Colors

| Role | Color |
|------|-------|
| Primary (buttons, links) | `#2d7a4f` Forest Green |
| Highlight (hover, selection) | `#4aab78` Light Green |
| Body background | `#ffffff` White |
| Sidebar background | `#eaf4ee` Mint Tint |

## Installation

1. Open GROWI Admin > Plugins
2. Enter the GitHub repository URL
3. Click Install and enable the plugin
4. Go to Admin > Customize > Theme and select **green**
